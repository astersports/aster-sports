  T0 · FREE BASIC HUB — public, no login, indexable. Search any AAU team, schedule,
       live scores, standings/records, tournament browse. FREE FOREVER (acquisition wedge).

       → $20/mo per ACCOUNT (every kid, every team) / $200/yr. (Price already in code.)
  [future] T1+ premium add-ons (film room, AI recaps) — define later (DR-6/film, AI).

PROGRAM LADDER (B2B — orgs sign on; "if not only parents like the app, the AAU programs
sign on") — Frank: define charging as we progress:
  P0 · CLAIMED LISTING — a program claims its public Hub presence (verified). Likely free
       (drives the directory + parent acquisition).
  P1 · PROGRAM HUB — the org maintains its own content: schedules, rosters, scores,
       announcements (the "maintaining their … schedules" tier). Subscription TBD.
  P2 · REGISTRATION + PAYMENTS — program registration/enrollment + fee collection (the
       existing register flow + financials). Per-registration fee or SaaS tier — TBD.
  [future] P3 communications/comms-engine, P4 advanced ops — define as adopted.
NOTE: the platform already carries `organizations.subscription_plan` (starter/pro/elite)
and a register/financials substrate — the program ladder reuses it; we wire the charging.

ENTITLEMENT MECHANICS (built): parent tier = `family_entitlements.status`; add a `plan`
column (free/plus) for future T1+ differentiation. `is_entitled = status in
(active,trialing)`. Program tier = `organizations.subscription_plan`. One Stripe-only
