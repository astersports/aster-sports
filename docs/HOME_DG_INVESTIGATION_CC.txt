HOW THE PARENT COMMS CARD LIGHTS UP
------------------------------------------------------------------------------
- Parent home "Needs you" comms item  ← useParentNeedsYou.commsItem
  ← useInboxList: SELECT FROM comms_message_recipients
       .eq('guardian_id', <the parent's guardianId>)
- No filter on delivery_status or sent_at. Any row whose guardian_id matches
  the parent shows up (null sent_at sorts to the top). So the ONLY thing that
  makes a parent's card light up is a comms_message_recipients row carrying
  THAT guardian's guardian_id.
