coach grouping, one Vercel-verified PR, no-regression guaranteed for the
single-program case.

  (a) src/hooks/useActivePrograms.js (NEW)
      The active program-season SET for the current user. `programs` is the

      program_type='season'), so this reads status='active' ACROSS

      nowMs). Home is off-season only when EVERY active program has ended. With
      one program this equals the prior isOffSeason(activeSeason,...) — the
      no-regression invariant. Wired into all three home pages in place of the
      singular gate.

  (c) src/components/home/CoachTail.jsx + src/lib/home/coachHomeData.js —
      groupTeamsByProgram(teams, programs). When the coach touches >1 active
      program, teams render GROUPED under per-program headers

      groups.length===1 and the flat render path is BYTE-UNCHANGED.
