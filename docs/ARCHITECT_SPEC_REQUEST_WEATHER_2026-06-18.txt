0. PURPOSE / THE RULING THAT CHANGES THE FRAME
   The #1071 decision-request leaned DEFER on a UTILITY argument (indoor basketball =
   low weather stakes). Frank's ruling supersedes that with an ENGAGEMENT argument:
   weather is a parent-delight touch worth having regardless of indoor/outdoor —
   "a cool parent ask." That reframe is legitimate and it CHANGES THE DESIGN BRIEF:
   the popover should be a pleasant micro-interaction, the rain banner should read
   FRIENDLY (a heads-up), not an alarm. Engagement, not operations. This aligns with
   the platform's own Nike-Run-Club engagement benchmark (CLAUDE.md §1) and the §16
   ELITE principles. CC is no longer arguing defer — this requests the render to
   build it WELL.

------------------------------------------------------------------------------
   - DELIGHT, not operations. A parent glancing at "Saturday's game" gets a small,

     the §16.14 detail-page hero/collapsible pattern if the popover ever expands.
